A framework that enables advanced Java mods for Project Zomboid.

[b]What it does:[/b]
ZombieBuddy is required by some mods to work. If a mod tells you to install ZombieBuddy, follow the instructions below.

[h1]Installation (One-Time Setup)[/h1]

[b]Step 1:[/b] Subscribe to ZombieBuddy on Steam Workshop

[b]Step 2:[/b] Installation is required ONLY ONCE for ZombieBuddy itself.

[b]🪟 Windows:[/b]

[u]Option A: Automated Installer (Recommended)[/u]
[list]
[*] Close Steam completely (right-click Steam in system tray → Exit)
[*] Download the latest ZombieBuddyInstaller.exe from [url=https://github.com/zed-0xff/ZombieBuddy/releases/]GitHub Releases[/url]
[*] Run the installer - it will automatically detect your game folder, copy the files, and set up Steam launch options.
[*] Start Steam.
[/list]

[u]Option B: Manual Installation[/u]
[list]
[*] Copy ZombieBuddy.jar and zbNative.dll from:
[code]C:\Program Files (x86)\Steam\steamapps\workshop\content\108600\3619862853\mods\ZombieBuddy\libs\[/code]
to your Project Zomboid folder:
[code]C:\Program Files (x86)\Steam\steamapps\common\ProjectZomboid\[/code]
[*] Open Project Zomboid's launch options in Steam
[*] Add this launch option: [code]-agentlib:zbNative --[/code]
[*] [b]⚠️ IMPORTANT:[/b] The -- at the end is mandatory - do not omit it!
[/list]

[b]🍎 macOS / 🐧 Linux:[/b]
[list]
[*] Copy ZombieBuddy.jar from the mod's libs/ directory to:
[list]
[*] [b]macOS:[/b] ~/Library/Application Support/Steam/steamapps/common/ProjectZomboid/Project Zomboid.app/Contents/Java/
[*] [b]Linux:[/b] ~/.steam/steam/steamapps/common/ProjectZomboid/projectzomboid/
[/list]
[*] Open Project Zomboid's launch options in Steam
[*] Add this JVM argument: -javaagent:ZombieBuddy.jar --
[*] [b]⚠️ IMPORTANT:[/b] The -- at the end is mandatory - do not omit it!
[/list]

[b]Step 3:[/b] After this setup, all mods that require ZombieBuddy will work automatically!

[h1]Uninstall[/h1]
[list]
[*] [b]Disable the agent first:[/b] Open Project Zomboid → Properties → clear the Launch Options line
[*] Delete zbNative.dll and ZombieBuddy.jar from your Project Zomboid install directory
[*] Unsubscribe from ZombieBuddy on Steam
[/list]

[h1]⚠️ Security Warning ⚠️[/h1]
Java mods enabled through ZombieBuddy have unrestricted access to your system. Only install mods from sources you trust.

[h1]FAQ[/h1]
[list]
    [*][b]Is it a virus?[/b]
    No.

    [*][b]How to start game in debug mode?[/b]
    [code]-agentlib:zbNative -- -debug[/code]

    [*][b]Do I need to repeat manual installation after I upgrade/downgrade/validate game?[/b]
    No.

    [*][b]Works on MP?[/b]
    Yes.

    [*][b]Save compatibility?[/b]
    Safe to add or remove on existing saves.

    [*][b]Supported game versions?[/b]
    B41, B42.12, B42.13, B42.14, B42.15, B42.16
[/list]
